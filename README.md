
# 世界灯火

> 愿中国青年都摆脱冷气，只是向上走。
>
> 不要听自暴自弃者的话。
>
> 能做事的做事，能发声的发声。
>
> 有一份光，发一份热。
>
> 就像萤火一般，也可以在黑暗里发一点光。
> 
> 不必等候炬火。
>
> 此后如竟没有炬火：我便是唯一的光。
> 
> 倘若有了炬火，出了太阳，我们自然心悦诚服的消失，不但毫无不平，而且还要随喜赞美这炬火或太阳；
> 
> 因为他照了人类，连我都在内。
> 
> 我又愿中国青年都只是向上走，不必理会这冷笑和暗箭。
>
>《鲁迅文集·杂文集·热风·随感录四十一》
